# Bishop's Media homepage starter

This is the front page starter for the new Josh Bishop photography/video website.

## Files
- `index.html` is the homepage.
- `styles/styles.css` contains all styling.
- `scripts/main.js` controls the mobile menu, sticky header, and optional b-roll video cycling.

## Asset placeholders to replace
Add Josh's actual files with these paths:
- `videos/home-broll-1.mp4`
- `videos/home-broll-2.mp4`
- `videos/home-broll-3.mp4`
- `images/hero-poster.jpg`
- `images/video-marketing.jpg`
- `images/real-estate-media.jpg`
- `images/events-weddings.jpg`
- `images/josh-bishop.jpg`

The three service cards link to pages that can be built next:
- `video-marketing-ads.html`
- `real-estate-media.html`
- `events-weddings.html`
## Revision note

The current homepage CSS has been revised toward a more minimalist style. Text glows, button shadows, heavy card shadows, and strong blue background glow effects were removed or toned down.
